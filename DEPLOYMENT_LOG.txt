OPENAI API KEY — DETAILED GENERATION STEPS
================================================================================

WHAT IT IS:
  Secret key from OpenAI that powers AI chat responses and embeddings.
  Format: sk-proj-... or sk-...

STEP-BY-STEP:
[ ] 1. Open https://platform.openai.com
[ ] 2. Sign up or log in
[ ] 3. Add billing (required for API usage):
        Profile -> Billing -> Add payment method
        OpenAI is pay-as-you-go (a few dollars covers testing)
[ ] 4. Open https://platform.openai.com/api-keys
[ ] 5. Click "Create new secret key"
[ ] 6. Name: Skincare-AI-Render
[ ] 7. Permissions: All (or Restricted with model access)
[ ] 8. Click Create -> COPY THE KEY IMMEDIATELY
        (You cannot view it again after closing the dialog)
[ ] 9. Paste into Render -> skincare-api -> Environment:
        OPENAI_API_KEY = sk-proj-xxxxxxxx...
[ ] 10. Never commit this key to GitHub

COST NOTE:
  - gpt-4o (set in render.yaml) costs more than gpt-4o-mini
  - Testing chat: typically pennies to a few dollars
  - Set usage limits at: platform.openai.com -> Settings -> Limits

IF YOU SKIP OPENAI_API_KEY:
  App still runs but chat uses basic fallback (less accurate).

================================================================================
ADMIN_API_KEY & JWT_SECRET — DETAILED GENERATION STEPS
================================================================================

ADMIN_API_KEY — your dashboard password
  Used when saving key in /dashboard/settings and for sync/upload APIs.

JWT_SECRET — backend signing key (different from ADMIN_API_KEY)
  Used internally to sign admin tokens. Never shown in the UI.

GENERATE ON WINDOWS (PowerShell) — run TWICE, use each output separately:
  -join ((48..57) + (65..90) + (97..122) | Get-Random -Count 40 | ForEach-Object {[char]$_})

OR use a password manager -> Generate 40-character random string.

ENTER IN RENDER BLUEPRINT (or skincare-api Environment):
  ADMIN_API_KEY = (your 40-char random string #1)
  JWT_SECRET    = (your 40-char random string #2)

SAVE ADMIN_API_KEY somewhere safe — you need it again at:
  https://skincare-frontend-z72h.onrender.com/dashboard/settings

DO NOT USE IN PRODUCTION:
  dev-admin-key-change-me
  dev-jwt-secret-change-me
